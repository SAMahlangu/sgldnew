# University SRC Governance System — Frontend

This is a small React + Vite scaffold containing a clean, accessible homepage for the University SRC Governance System. It is ready to be integrated with Supabase or Firebase later.

Quick start

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

Files of interest

- `src/App.jsx` — homepage layout and content
- `src/components/RoleCard.jsx` — role card component
- `src/styles.css` — basic styling and accessibility helpers

Next steps

- Add authentication with Supabase or Firebase and wire `Login` buttons to auth flows.
- Implement dashboards for each user role.
